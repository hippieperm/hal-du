## 관리자 로그인 방법

### 1. 회원가입 (최초 사용 시)
1. 웹사이트 상단 메뉴에서 **"회원가입"** 클릭
2. 회원가입 폼에서:
   - **이메일**: `admin@haldo.kr` 또는 `team@haldo.kr` 입력
   - **비밀번호**: 6자리 이상의 안전한 비밀번호 설정
   - **이름**: 관리자명 입력
   - **전화번호**: 연락 가능한 번호 입력
3. 약관 동의 후 **"가입하기"** 클릭

### 2. 로그인
1. 웹사이트 상단 메뉴에서 **"로그인"** 클릭
2. 관리자 이메일과 비밀번호 입력
3. **"로그인"** 버튼 클릭

## 상품 관리 기능

관리자로 로그인하면 Shop 페이지에서 다음 기능을 사용할 수 있습니다:

### 1. 상품 추가
1. **Shop** 페이지로 이동
2. 페이지 상단 우측의 **"상품 추가"** 버튼 클릭
3. 상품 정보 입력:
   - **상품명**: 상품의 이름
   - **가격**: 숫자만 입력 (원 단위)
   - **설명**: 상품 상세 설명
   - **이미지 URL**: 상품 이미지 주소
   - **카테고리**: 상품 분류
   - **재고**: 재고 수량
4. **"등록"** 버튼 클릭하여 상품 저장

### 2. 상품 수정
1. **Shop** 페이지에서 수정할 상품 찾기
2. 상품 카드를 **길게 누르기** (Long Press)
3. 편집 대화상자가 나타나면 정보 수정
4. **"수정"** 버튼 클릭하여 변경사항 저장

### 3. 상품 삭제
1. **Shop** 페이지에서 삭제할 상품 찾기
2. 상품 카드를 **길게 누르기** (Long Press)
3. 편집 대화상자에서 **"삭제"** 버튼 클릭
4. 확인 대화상자에서 **"삭제"** 재확인

### 4. 관리자 표시
- 관리자로 로그인 시 상품 카드 우측 상단에 **편집 아이콘(✏️)** 표시
- 이 아이콘으로 관리자 모드임을 확인 가능

## 주의사항

### ⚠️ 보안 관련
- 관리자 비밀번호는 안전하게 관리하세요
- 공용 컴퓨터에서는 작업 후 반드시 로그아웃하세요
- 관리자 계정 정보를 타인과 공유하지 마세요

### ⚠️ Firebase 설정
- Firebase Console에서 **Authentication** 활성화 필요
- **이메일/비밀번호** 인증 방식이 켜져 있어야 함
- Firebase 프로젝트: `haldu-webapp`

### ⚠️ 데이터 관리
- 상품 삭제는 복구가 불가능하므로 신중하게 진행
- 상품 이미지는 유효한 URL이어야 함
- 가격 입력 시 숫자만 입력 (콤마, 원 표시 자동 처리)

### ⚠️ 브라우저 호환성
- Chrome, Edge, Safari 최신 버전 권장
- 모바일에서는 길게 누르기 대신 터치 유지

## 문제 해결

### 관리자 권한이 표시되지 않을 때
1. 로그인한 이메일이 관리자 이메일 목록에 있는지 확인
2. 브라우저 새로고침 (F5 또는 Cmd+R)
3. 로그아웃 후 다시 로그인

### 상품이 저장되지 않을 때
1. 모든 필수 필드가 입력되었는지 확인
2. 이미지 URL이 유효한지 확인
3. 네트워크 연결 상태 확인

### Firebase 오류 발생 시
1. Firebase Console에서 프로젝트 상태 확인
2. Authentication 서비스 활성화 여부 확인
3. 할당량 초과 여부 확인
